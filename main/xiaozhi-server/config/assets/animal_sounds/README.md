# 动物叫声音频文件目录

此目录用于存放动物叫声的音频文件，用于"动物互动"模式。

## 文件命名规范

根据情绪类型，音频文件应命名为：

- `cat_happy.wav` - 开心情绪（对应关键词：开心、高兴、快乐、喜悦）
- `cat_sad.wav` - 悲伤情绪（对应关键词：悲伤、难过、伤心、沮丧）
- `cat_angry.wav` - 生气情绪（对应关键词：生气、愤怒、气愤）
- `cat_afraid.wav` - 害怕情绪（对应关键词：害怕、恐惧、紧张）
- `cat_helpless.wav` - 无奈情绪（对应关键词：无奈、唉、叹气）
- `cat_coquetry.wav` - 撒娇情绪（对应关键词：撒娇、卖萌、黏人）
- `cat_neutral.wav` - 默认情绪（当无法匹配到其他情绪时使用）

## 音频格式要求

- 建议格式：WAV 或 MP3
- 采样率：16kHz
- 声道：单声道（mono）
- 比特率：建议 16-bit

## 使用说明

1. 将准备好的音频文件放入此目录
2. 在智控台的TTS模型配置中选择"动物叫声"模型
3. 确保LLM输出的文本中包含情绪关键词，系统会自动匹配对应的音频文件
4. 如果找不到对应情绪的音频，会使用默认音频文件（`cat_neutral.wav`）

## 自定义配置

可以在智控台的模型配置中自定义：
- 音频文件路径（`base_path`）
- 情绪到音频文件的映射（`emotion_files`）
- 情绪关键词映射（`emotion_keywords`）

详细配置说明请参考：`docs/animal-sound-mode.md`
