TTS Test Quick Start Guide
==========================

1. Set API Keys (one-time setup)
---------------------------------

Add to ~/.zshrc for permanent access:

echo 'export DEEPGRAM_API_KEY="sk-your-key"' >> ~/.zshrc
echo 'export ELEVENLABS_API_KEY="sk-your-key"' >> ~/.zshrc  
echo 'export CARTESIA_API_KEY="your-key"' >> ~/.zshrc
echo 'export MINIMAX_API_KEY="your-key"' >> ~/.zshrc
echo 'export MINIMAX_GROUP_ID="your-group-id"' >> ~/.zshrc

# FishSpeech
echo 'export FISH_API_KEY="your-key"' >> ~/.zshrc
echo 'export FISH_REFERENCE_ID="reference-id"' >> ~/.zshrc  # optional

source ~/.zshrc

4. Troubleshooting
------------------

If SDK not installed:
  pip install deepgram-sdk elevenlabs cartesia aiohttp fish-audio-sdk

If API key error:
  echo $DEEPGRAM_API_KEY        # Check if set
  echo $MINIMAX_API_KEY         # Check Minimax key
  echo $MINIMAX_GROUP_ID        # Check Minimax group ID
  echo $FISH_API_KEY            # Check FishSpeech key
  source ~/.zshrc               # Reload config

If module not found:
